6.0 Create an Extract
After at least one Source and one Target is configured, you can then setup an Extract. Extracts tie together a Source and Target.

To configure the site, type the following URL for the Administrative Interface into your browser: http://{SITE_URL}/setup From here, you should see a menu that includes an option to create an Extract. You will be asked to assign an EXTRACT_NAME to each Extract. Creating a new Extract will create a file at {INSTALLATION_DIRECTORY} / {EXTRACTS_DIRECTORY} . This directory contains multiple files, one for each download job.

The Extract Configuration (Configuration Guide) presents detailed information on Extract configuration.

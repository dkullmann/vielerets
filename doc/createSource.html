4.0 Create a Source
The first step to configuring the installation is to setup a connection to a data source. A data sources is typically a Multiple Listing Service (MLS) operating a RETS server.

To configure the site, type the following URL for the Administrative Interface into your browser: http://{SITE_URL}/setup From here, you should see a menu that includes a setup option. When you enter the Setup, you should see options for both Sources and Displays. You will be asked to assign a SITE_NAME to each Source. Defining a new Source will create a single file at {INSTALLATION_DIRECTORY} / {SOURCES_DIRECTORY} /{SITE_NAME}.

The Source Configuration (Configuration Guide) presents detailed information on package configuration.

1.0 Dependencies
This package is a group of PHP classes a web server that work together to support a displays that have HTML Version 4 compliant tags. No database or local repository is required. The package does not use plug-ins, Java Applets, ActiveX components or Javascript to reduce exposure to security exploits.

The package is designed to support the following browsers:
  • Internet Explorer
  • Konquerer
  • Mozilla (and derivatives)
  • Netscape
  • Safari
It has been reported to operate with PHP4 and PHP5 installed on the following web servers:
  • IIS
  • Apache
  • OmniHTTPd
This list is not complete though. Any web server that supports PHP4 or PHP5 should be supported. The installation of PHP is beyond the scope of this document but is available from: http://www.php.net If you are using a version of PHP that is 4.3 or higher, you should set the default_socket_timeout to something greater than the default of 60 seconds (perhaps 240) in your php.ini file if you will be hitting a particularly slow server.

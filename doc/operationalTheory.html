1.0 Operational Theory
This package allows you to download listings from a RETS server and store them in a variety of formats. It is coded in the popular PHP language.

One of the most common approaches used to collecting information that supports consumer websites is to download data and images from a Multiple Listing Service (MLS). This information is then stored in a repository that is used to support consumer inquiries. The local repository is then refreshed on a periodic basis. The term we will use for this style will be Persistent Download. A graphic that depicts the elements of this approach follows.
